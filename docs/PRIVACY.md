# AutoVerdict — 개인정보 처리방침

**최종 업데이트: 2026-04-09**

AutoVerdict(이하 "본 확장")은 엔카(Encar) 중고차 매물 페이지에서 사용자가 검토 중인 매물 정보를 자동으로 수집·분석하여 체크리스트 형태의 위험 신호를 제공하는 Chrome 확장 프로그램입니다. 본 확장은 사용자의 개인정보를 최소한으로만 접근하며, 수집한 데이터는 **원칙적으로 사용자의 로컬 브라우저를 벗어나지 않습니다**.

## 1. 수집하는 정보

본 확장은 다음 정보를 **현재 브라우저 탭이 엔카 매물 상세 페이지일 때에만** 수집합니다.

| 카테고리 | 데이터 | 출처 |
|---|---|---|
| 매물 기본 정보 | 차량 제조사/모델/연식/주행거리/가격/옵션 | 엔카 API 응답 |
| 차량 이력 | 소유자 변경 횟수, 보험 처리 이력, 자차보험 공백 구간 | 엔카 API 응답 |
| 사고 이력 | 자차 / 타차 보험 처리 내역, 피해 금액 | 엔카 API 응답 |
| 성능·상태 점검 | 외판/주요 골격 상태, 프레임 신호 | 엔카 성능점검 리포트 |
| 평가 결과 | 위 데이터로 로컬에서 계산한 체크리스트 결과 및 점수 | 본 확장이 생성 |

**수집하지 않는 것**: 사용자의 엔카 계정 정보(이메일/비밀번호), 결제 정보, 다른 웹사이트의 쿠키, 브라우저 기록, 사용자 식별자 등은 수집하지 않습니다.

## 2. 데이터 저장 위치

수집된 모든 데이터는 **사용자의 로컬 브라우저 내부(IndexedDB, chrome.storage.local)** 에만 저장됩니다. 본 확장 개발자의 서버나 제3자 서버로 **전송되지 않습니다**.

저장 기간은 사용자가 확장을 제거하거나 Chrome 저장소를 초기화할 때까지입니다.

## 3. 외부 전송 — AI 평가 기능을 사용할 때만

본 확장의 "AI 평가" 기능을 사용하려면, 사용자가 **직접 자신의 OpenAI 또는 Google AI Studio API 키를 입력**해야 합니다. 이 기능을 사용하면:

- 사용자가 입력한 API 키는 **메모리(React state)에만** 저장되며, localStorage/IndexedDB/chrome.storage 어디에도 기록되지 않습니다. 사이드패널을 닫거나 새로고침하면 다시 입력해야 합니다.
- 평가 실행 시, 수집한 매물 데이터(차량 사양/이력/점검 결과)가 사용자의 API 키와 함께 **해당 LLM 공급자(OpenAI 또는 Google)의 API 엔드포인트로 직접 전송**됩니다. 본 확장 개발자의 서버를 경유하지 않습니다.
- 전송된 데이터에 대한 LLM 공급자의 처리 방침은 각 서비스의 개인정보 처리방침을 따릅니다.
  - [OpenAI Privacy Policy](https://openai.com/policies/privacy-policy)
  - [Google AI Privacy Notice](https://ai.google.dev/terms)

AI 평가 기능을 사용하지 않으면 어떠한 데이터도 외부로 나가지 않습니다.

## 4. 권한 사용 내역

Chrome 확장 `manifest.json`에 선언된 각 권한의 사용 목적은 다음과 같습니다.

| 권한 | 사용 목적 |
|---|---|
| `storage` | 매물 평가 결과를 로컬에 캐싱 (재방문 시 재계산 생략) |
| `sidePanel` | 평가 결과를 사이드패널 UI로 표시 |
| `cookies` | 엔카 API 호출 시 사용자의 엔카 로그인 세션 사용 (별도 인증 수집 없음) |
| `alarms` | 캐시 만료 주기 관리 |
| `scripting` | 엔카 매물 상세 페이지에 파싱 스크립트 주입 |
| `tabs` | 현재 탭이 엔카 매물 상세 URL인지 감지 |
| `host_permissions`(`fem.encar.com`, `car.encar.com`, `api.encar.com`) | 엔카 도메인에서만 데이터 수집 (다른 사이트는 접근 불가) |

## 5. 제3자 공유

본 확장은 수집한 데이터를 제3자에게 **판매/공유/광고 목적으로 제공하지 않습니다**. 유일한 외부 전송은 위 §3에 명시된, 사용자가 명시적으로 AI 평가를 실행할 때의 LLM API 호출뿐입니다.

## 6. 사용자의 권리

- 저장된 모든 평가 데이터는 Chrome의 확장 관리 화면에서 본 확장을 제거하면 즉시 함께 삭제됩니다.
- 특정 매물의 캐시만 삭제하려면 사이드패널의 "↻ 재평가" 버튼으로 덮어쓸 수 있습니다.
