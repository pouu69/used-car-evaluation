# AutoVerdict

엔카(Encar) 중고차 매물을 11가지 핵심 룰로 자동 점검하는 Chrome 확장 프로그램.

## 문서

- [개인정보 처리방침 (Privacy Policy)](./PRIVACY)
- [Chrome Web Store 등록 가이드](./CHROME_WEB_STORE)

## 주요 기능

- 사고 / 프레임 / 보험 이력 / 시세 비교 11개 룰 자동 점검
- Brutalist Scoreboard UI — Archivo Black + 형광 옐로우 기반
- (선택) OpenAI / Gemini API 키를 이용한 AI 종합 평가

## 개인정보

수집한 모든 매물 데이터는 사용자의 로컬 브라우저에만 저장되며, 확장 개발자의 서버는 존재하지 않습니다. 외부 전송은 사용자가 직접 AI 평가를 실행할 때의 LLM API 호출이 유일하며, 자세한 내용은 [개인정보 처리방침](./PRIVACY)을 확인하세요.
